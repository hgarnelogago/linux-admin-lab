# Linux Admin Lab

Repositorio personal para documentar mi aprendizaje de Linux como administrador de sistemas.

Estoy usando una máquina virtual con Ubuntu Server para practicar administración por terminal, SSH, usuarios, permisos, servicios, red, logs, Bash scripting y Docker.

## Entorno

- Sistema operativo: Ubuntu Server
- Virtualización: VMware
- Acceso remoto: SSH
- Objetivo: aprender administración Linux desde terminal

## Temas

- Terminal básica
- Estructura de directorios
- Usuarios, grupos y permisos
- Gestión de paquetes con APT
- Servicios con systemd
- Red y SSH
- Logs y troubleshooting
- Bash scripting
- Docker
- Seguridad básica
