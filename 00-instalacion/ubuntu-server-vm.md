# Instalación de Ubuntu Server en VMware

## Objetivo

Crear una máquina virtual con Ubuntu Server para practicar administración de sistemas Linux desde terminal.

## Entorno usado

- Hipervisor: VMware
- Sistema invitado: Ubuntu Server
- Acceso remoto: SSH

## Pasos realizados

1. Crear una máquina virtual en VMware.
2. Instalar Ubuntu Server.
3. Crear usuario administrador.
4. Activar SSH.
5. Conectarme desde mi equipo principal por SSH.
